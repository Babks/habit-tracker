# Habit Tracker

Habit Tracker — это backend-приложение для отслеживания привычек пользователя.

Проект разработан на Spring Boot и предназначен для практики построения REST API, работы с базой данных и реализации бизнес-логики.

---

## Цель проекта

Основная цель проекта:

- Практика разработки backend-приложения на Spring Boot
- Построение чистой архитектуры (Controller -> Service -> Repository)
- Работа с базой данных через Spring Data JPA
- Реализация бизнес-логики подсчёта streak (серии выполнения привычек)
- Практика валидации данных и обработки ошибок

---

## Функциональность

Планируемый функционал:

- Авторизация пользователя (JWT)
- Создание привычки
- Редактирование привычки
- Удаление привычки
- Получение списка привычек
- Отметка выполнения привычки
- Подсчёт текущей серии выполнения (streak)
- Получение статистики выполнения за период
- Фильтрация привычек по типу и статусу

Дополнительно:

- Роли (USER / ADMIN)
- Геймификация (баллы за выполнение)
- Напоминания через Scheduler
- Docker-контейнеризация

---

## Архитектура проекта

Проект будет построен по слоистой архитектуре:

- Controller — обработка HTTP-запросов
- Service — бизнес-логика
- Repository — работа с базой данных
- Entity — модели данных
- DTO — объекты передачи данных
